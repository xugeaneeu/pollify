#### **Вводимые данные:**

1. При создании опроса:
- название опроса
- описание
- текст вопроса
- варианты ответов
- тип опроса:
  - анонимный / неанонимный
  - одиночный / множественный
- максимальное количество вариантов (для множественного выбора)
- разрешение пользовательского ответа
- дата начала и окончания

2. При голосовании:
- один или несколько option_id (для выбора)
- или текстовый ответ (custom_text)

3. При подаче жалобы:
- идентификатор опроса
- причина жалобы
- комментарий

4. При модерации:
- решение администратора (APPROVE / REJECT)

---

#### **Изменяемые данные:**

1. Параметры опроса (до начала голосования):
- название
- описание
- сроки проведения

2. Состояние опроса:
- скрыт (is_hidden)
- активен / завершён (по времени)

3. Жалобы:
- статус жалобы
- решения администраторов

---

#### **Удаляемые данные:**  

- опросы (ограниченно, предпочтительно скрытие)
- варианты ответов (при удалении опроса)
- голоса (при удалении опроса)
- жалобы (при удалении опроса)
- технические данные (логи)

---

#### **Вычисляемые данные:**

- общее количество голосов
- количество голосов по вариантам
- процентное распределение
- статус опроса (по датам)
- итог по жалобе (на основе кворума)
- количество голосов администраторами по жалобе

---

#### **Справочные данные:**  

- статусы опросов (активен, завершён, скрыт)
- типы опросов:
  - анонимный / неанонимный
  - одиночный / множественный
- статусы жалоб (OPEN, IN_REVIEW, RESOLVED, REJECTED)
- решения администраторов (APPROVE, REJECT)
- роли пользователей (USER, ADMIN)
- коды ошибок API
