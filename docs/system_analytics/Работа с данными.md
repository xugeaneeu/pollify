#### **Вводимые данные:**
Данные, которые вводятся пользователями или передаются через API:
1. При создании опроса:
	- название опроса
    - описание
    - текст вопроса
    - варианты ответов
    - тип опроса (анонимный / не анонимный)
    - дата начала и окончания
    - параметры доступа
2. При голосовании:
    - идентификатор выбранного варианта ответа

#### **Изменяемые данные:**
Данные, которые могут быть изменены после создания:
1.  Параметры опроса (до начала голосования):
    - название
    - описание
    - сроки проведения
    - настройки доступа
2. статус опроса:
    - активен
    - завершён

#### **Удаляемые данные:**  
Данные, которые могут быть удалены из системы:
1. Опросы (до начала или при административном управлении)
2. Результаты голосования (например, при удалении опроса)
3. Технические данные (временные записи, логи)

#### **Вычисляемые данные:**
Данные, которые не хранятся напрямую, а формируются системой:
1. общее количество голосов
2. количество голосов по каждому варианту
3. процентное соотношение голосов
4. статус опроса (активен / завершён — может вычисляться по датам)

#### **Справочные данные:**  
Относительно постоянные данные, используемые системой:
1. статусы опросов (активен, завершён, запланирован)
2. типы опросов (анонимный, не анонимный)
3. роли пользователей
4. коды ошибок и статусы API
