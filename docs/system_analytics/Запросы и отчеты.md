#### **Результаты голосования:**  
Запрос на получение итогов опроса:
- общее количество голосов (с учётом множественного выбора)
- распределение по вариантам
- процентное соотношение
- список текстовых ответов (если разрешены)
- (в неанонимном режиме — при наличии прав) детализация по пользователям

#### **Статистика по вариантам:**  
Более детализированная аналитика:
- количество голосов по каждому варианту
- процент по каждому варианту
- сравнение вариантов между собой
- динамика голосования (опционально)

#### **Списки опросов:**  
Запросы на получение перечня опросов:
- все опросы
- опросы, созданные пользователем
- доступные для голосования опросы

С возможной фильтрацией:
- по дате
- по статусу (активен / завершён / скрыт)
- по типу:
  - анонимный / неанонимный
  - одиночный / множественный
  - с текстовыми ответами

#### **Активные / завершённые опросы:**  
Отдельные выборки:
- активные опросы:
    - доступны для голосования
    - текущая дата в пределах периода проведения
    - не скрыты модерацией
- завершённые опросы:
    - голосование закрыто
    - доступны результаты

#### **Жалобы и модерация:**
- список жалоб
- жалобы по конкретному опросу
- статус жалобы (OPEN, IN_REVIEW, RESOLVED, REJECTED)
- решения администраторов по жалобе
- итоговое решение (на основе кворума)
