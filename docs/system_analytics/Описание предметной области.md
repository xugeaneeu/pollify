Система голосования/сбора мнений предназначена для организации опросов с возможностью гибкой настройки и безопасной модерации.

#### Текущий процесс:
Опросы проводятся:
- вручную
- через мессенджеры
- через разрозненные сервисы

Недостатки:
- нет централизованного хранения
- слабая защита от накрутки
- сложность анализа
- отсутствие прозрачной модерации

---

#### Основная бизнес-функция:
Сбор голосов пользователей

---

#### Бизнес-процессы:

### 1. Создание опроса
1. Пользователь проходит авторизацию
2. Переходит к созданию опроса
3. Вводит:
   - название
   - описание
   - вопрос
   - варианты
4. Настраивает:
   - анонимность
   - тип голосования:
     - одиночный
     - множественный (с ограничением)
   - возможность текстового ответа
   - сроки
5. Система валидирует данные
6. Сохраняет опрос

---

### 2. Голосование
1. Пользователь авторизуется
2. Выбирает опрос
3. Система проверяет доступ и участие
4. Пользователь:
   - выбирает варианты
   - или вводит текст
5. Система:
   - фиксирует участие
   - сохраняет голос(а)
   - соблюдает ограничения (max_choices)

---

### 3. Просмотр результатов
1. Пользователь выбирает опрос
2. Система проверяет доступ
3. Формирует результаты:
   - агрегированные данные
   - текстовые ответы (если есть)

---

### 4. Модерация (через жалобы)
1. Пользователь подаёт жалобу
2. Жалоба регистрируется
3. Администраторы:
   - просматривают жалобу
   - принимают решения
4. При достижении кворума:
   - опрос скрывается или остаётся
