### Инициатор опроса (создатель):
Пользователь, создающий и настраивающий опросы.
**Действия:**
- создает опросы через API
- задаёт параметры:
    - вопрос и варианты ответов
    - анонимность
    - сроки проведения
    - ограничения доступа
- публикует опрос
- просматривает результаты
- (опционально) управляет опросом — редактирует или завершает

### Участник голосования (пользователь):
Пользователь, принимающий участие в опросах.
**Действия:**
- получает список доступных опросов
- просматривает содержимое опроса
- голосует (1 раз на опрос)
- получает подтверждение голосования
- просматривает результаты (если разрешено)
