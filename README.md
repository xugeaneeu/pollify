# Pollify

## Описание
REST-сервис с GUI клиентом для проведения опросов с поддержкой:
- анонимного и неанонимного голосования
- множественного выбора
- пользовательских текстовых ответов
- модерации через жалобы с кворумом администраторов

## Основные возможности
- создание и управление опросами
- голосование с ограничением «один пользователь — один опрос»
- просмотр результатов
- подача жалоб на опросы
- распределённая модерация

## Архитектура
- Клиент (frontend)
- REST API
- PostgreSQL:18

## Особенности
- анонимность достигается отсутствием связи user → vote
- поддержка разных типов голосования в одной модели
- модерация без единоличного контроля

## Ограничения
- нельзя одновременно включить множественный выбор и текстовый ответ
- администратор не может напрямую изменять данные голосования

## Стек
- PostgreSQL 18
- REST API
